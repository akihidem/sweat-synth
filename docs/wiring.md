# 配線 — 手汗シンセ

> ⚠️ 物理的なはんだ付け・組み立て(=電子工作)は本リポの範囲外。
> ここは「どう繋ぐか」の設計記録。実際の接続は各自で。

## 接続図(GSR モジュール経由)

```
   指/手のひら
   ┌───────┐
   │ 電極A │───┐
   └───────┘   │   ┌──────────────┐        ┌──────────────────┐
               ├──▶│ GSR モジュール│  OUT  │ XIAO nRF52840     │
   ┌───────┐   │   │ (Grove等)     │──────▶│ A0 (ADC, 10bit)   │
   │ 電極B │───┘   │  VCC ◀────────┼───────│ 3V3               │
   └───────┘       │  GND ◀────────┼───────│ GND               │
                   └──────────────┘        │                   │
                                            │  BLE-MIDI ))) ────┼──▶ iPad / PC
   ┌──────────┐                             │                   │
   │LiPo 100mA│── TP4056 ──── BAT+/BAT- ───▶│ (任意:モバイル化)  │
   └──────────┘                             └──────────────────┘
```

## ピン対応

| XIAO ピン | 接続先 | 備考 |
|-----------|--------|------|
| `A0`  | GSR モジュール OUT | ADC 入力。ファームの `ADC_PIN` と一致 |
| `3V3` | GSR モジュール VCC | センサ電源 |
| `GND` | GSR モジュール GND | 共通グランド |
| `BAT+ / BAT-` | LiPo(任意) | XIAO 裏面パッド。モバイル運用時のみ |

## キャリブレーション
- 起動直後は適応レンジが立ち上がるまで数〜数十秒かかる(`RANGE_RELEASE_SEC`)。
- 手をリラックス→緊張、と一度往復させるとレンジが馴染む。
- 絶対 µS は校正していない(相対変化で鳴らす設計)。正確な µS が要るなら
  既知抵抗を電極間に挟んで OUT 電圧を 2 点測り、ファームの `readEda()` に線形換算を入れる。

## 自作回路にする場合(GSR モジュールを使わない)
- 定電圧を電極へかけ、直列基準抵抗との分圧を ADC で読む(簡易 GSR)。
- 接触圧と乾燥に弱いので、まずはモジュールで動かしてから移植推奨。
